# VoidZip - File Compressor

*VoidZip is a modern desktop file compression and extraction tool built with Python and CustomTkinter.*  
*It features a sleek black-themed interface with fast ZIP compression, clean UI design, and simple file management.*

---

## Preview

A minimal and modern file compression utility with:
- ZIP file compression
- ZIP extraction
- Dark premium UI
- Fast local processing
- Simple and clean experience

---

## Features

- Compress files into ZIP archives
- Extract ZIP files instantly
- Modern black UI with neon green accents
- Beginner-friendly and lightweight
- Modular project structure

## Installation

### 1. Clone the repository

```bash

```

### 2. Navigate to project folder

```bash
cd VoidZip
```

```bash

```

## How It Works

### Compression
- Select a file
- Click **Compress**
- The compressed ZIP file will be saved inside the `output/` folder

### Extraction
- Select a ZIP file
- Click **Extract ZIP**
- Extracted contents will be saved inside the `output/` folder
